# 🚗 Managementul unui Parc Auto cu Licitații Auto

Proiect dezvoltat pentru cursul de Baze de Date.

## 📌 Descriere
Acest proiect reprezintă proiectarea și implementarea unei baze de date relaționale destinate gestionării fluxului operațional al unei platforme online de licitații auto. Sistemul centralizează inventarul vehiculelor, alocarea mecanicilor pentru inspecții, istoricul daunelor și desfășurarea licitațiilor cu oferte în timp real.

## 🛠️ Tehnologii Utilizate
* **Bază de Date:** Oracle SQL
* **Mediu de dezvoltare:** JetBrains DataGrip
* **Interfață Web:** Oracle APEX 

## 📂 Structura Repository-ului
* 📄 **Documentație PDF (`proiectBDIonitaRares.pdf`):** Conține descrierea detaliată a entităților, diagramele ERD și conceptuale, precum și explicarea tuturor regulilor de business și a constrângerilor de integritate.
* 💻 **Script SQL:** Fișierul `.sql` care conține codul complet pentru crearea structurii bazei de date (DDL), popularea cu date de test (DML) și definirea vizualizărilor (Views).
